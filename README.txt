==============================
== Installation Instructions
==============================
Download python rest controller from http://code.google.com/p/python-rest-client/downloads/list
Put it in the same directory as Unassembla, under 'python-rest-client'

==============================
== Usage
==============================

Modify settings in settings.py for the login and project details, along
with the mapping of users. Then just run using python.

"Python unfuddle_to_assembla.py"

Voila!
